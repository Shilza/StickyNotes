# Install
1. git clone https://github.com/Shilza/StickyNotes
2. npm install
3. npm run start (backend)
4. cd frontend
5. npm run start (frontend)

# Stack
* React
* Styled-components
* Effector
* GraphQL (Apollo)
* NodeJs (Express)
* MongoDB

# Features
* Authentication
* Boards creation
* Customise boards
* Lists creation
* Records creation
* Adding marks to records
* Lists and records drag and drop
